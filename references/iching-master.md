# 易经与梅花易数宗师级参考手册 v3.0

## 第一性原理：易经的底层逻辑

易经的本质是**变化的数学**。宇宙的一切变化，都可以用阴阳两种力量的组合来描述。64卦是64种基本的变化模式，每一卦都是一个完整的故事——从起始状态，到发展过程，到最终走向。

**核心原则**：
1. **变是唯一的不变**：每一卦都在描述一个变化的过程，不是静态的状态
2. **时位**：同一卦在不同时间（爻位）有不同含义，时机比内容更重要
3. **象数理**：象（图像）→数（数字）→理（道理），三层解读缺一不可
4. **体用**（梅花易数）：体卦是自己，用卦是对方/外界，体克用则吉，用克体则凶

## 梅花易数完整起卦方法

### 方法一：时间起卦（最常用）

**年月日时起卦**：
- 上卦 = （年数 + 月数 + 日数）÷ 8，取余数（余0为8）
- 下卦 = （年数 + 月数 + 日数 + 时辰数）÷ 8，取余数（余0为8）
- 动爻 = （年数 + 月数 + 日数 + 时辰数）÷ 6，取余数（余0为6）

**年数计算**：取农历年天干地支数之和
- 天干数：甲1、乙2、丙3、丁4、戊5、己6、庚7、辛8、壬9、癸10
- 地支数：子1、丑2、寅3、卯4、辰5、巳6、午7、未8、申9、酉10、戌11、亥12

**时辰数**：子1、丑2、寅3、卯4、辰5、巳6、午7、未8、申9、酉10、戌11、亥12

**示例**：2024年（甲辰年）3月15日午时起卦
- 年数 = 甲(1) + 辰(5) = 6
- 月数 = 3
- 日数 = 15
- 时辰数 = 午(7)
- 上卦 = (6+3+15) ÷ 8 = 24 ÷ 8 = 3，余0，取8 → 坤卦
- 下卦 = (6+3+15+7) ÷ 8 = 31 ÷ 8 = 3余7，取7 → 艮卦
- 动爻 = (6+3+15+7) ÷ 6 = 31 ÷ 6 = 5余1，取1 → 初爻动
- 本卦：坤上艮下 = 剥卦，初爻动

### 方法二：数字起卦

取任意数字（如报纸页码、随机数等）：
- 上卦 = 数字 ÷ 8，取余数
- 下卦 = 数字 ÷ 8，取余数（如只有一个数字，则再取一个）
- 动爻 = 数字 ÷ 6，取余数

### 方法三：物象起卦

看到某个物象，根据其属性对应八卦：
- 天、圆形、金属 → 乾
- 水、低洼、鱼 → 坎
- 山、石头、门 → 艮
- 雷、木、绿色 → 震
- 风、绳、长形 → 巽
- 火、光、网 → 离
- 泽、水边、口 → 兑
- 地、方形、布 → 坤

## 体用卦判断（核心）

### 体用卦确定规则
- **问自己的事**：上卦为用，下卦为体
- **问他人的事**：上卦为体，下卦为用
- **动爻所在卦为用卦**（重要！）

### 体用五行生克判断

| 关系 | 含义 | 吉凶 |
|-----|-----|-----|
| 用生体 | 外界/对方帮助我 | 大吉 |
| 体生用 | 我帮助外界/对方 | 小吉，但我有付出 |
| 体克用 | 我克制外界/对方 | 吉，我有主导权 |
| 用克体 | 外界/对方克制我 | 凶，我处于被动 |
| 体用比和 | 同五行，势均力敌 | 平，视具体情况 |

## 互卦与变卦（高级推演）

### 互卦（本卦的内在结构）
取本卦2、3、4爻为下互卦，3、4、5爻为上互卦，组成互卦。
互卦代表事情发展的中间过程，是隐藏的力量。

**示例**：乾卦（111111）
- 下互卦：2、3、4爻 = 111 = 乾
- 上互卦：3、4、5爻 = 111 = 乾
- 互卦：乾上乾下 = 乾卦（乾卦互卦还是乾，极阳之象）

### 变卦（动爻变化后的结果）
动爻变化：阳爻变阴，阴爻变阳，得到变卦。
变卦代表事情的最终走向和结果。

**解读顺序**：本卦（现状）→ 互卦（过程）→ 变卦（结果）

## 动爻解读规则

### 动爻的重要性
动爻是卦中最活跃的力量，代表变化的核心。

### 六爻位置含义
| 爻位 | 象征 | 含义 |
|-----|-----|-----|
| 初爻（最下） | 地下、潜伏 | 事情的起始，潜伏阶段 |
| 二爻 | 地上、民间 | 事情的发展，民间层面 |
| 三爻 | 人位下 | 过渡，容易有变数 |
| 四爻 | 人位上 | 接近权力，需要谨慎 |
| 五爻 | 天位下 | 最佳位置，君位 |
| 上爻（最上） | 天上、极端 | 物极必反，过犹不及 |

### 多爻动的处理
- 一爻动：以动爻为主，看动爻的含义
- 两爻动：以上爻为主（上爻代表结果）
- 三爻动：以互卦为主
- 四爻动：以不动的两爻为主
- 五爻动：以不动的一爻为主
- 六爻动：乾坤两卦看用九用六，其余卦看变卦

- 体用相生之时，事情顺利
- 体用相克之时，有冲突或阻碍
