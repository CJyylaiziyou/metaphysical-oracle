# 塔罗小阿卡纳完整牌义（56张）

## 权杖（火象：行动/事业/激情/意志）

| 牌名 | 正位含义 | 逆位含义 | 人话版 |
|-----|---------|---------|-------|
| 权杖A | 新开始、灵感爆发、行动信号 | 时机未到、缺乏动力 | 正：冲！现在是行动的时候。逆：还没准备好，先等等。 |
| 权杖2 | 规划未来、等待结果、有远见 | 犹豫不决、计划受阻 | 正：你已经出发了，现在等结果。逆：计划还没想清楚。 |
| 权杖3 | 扩张、等待回报、前景广阔 | 延迟、障碍、需要调整 | 正：你播的种快要发芽了。逆：结果来得比预期慢。 |
| 权杖4 | 庆祝、稳定、家园、成就 | 不稳定、缺乏归属感 | 正：值得庆祝的时刻，享受它。逆：表面喜庆，内心不踏实。 |
| 权杖5 | 竞争、冲突、混乱 | 避免冲突、内部分歧 | 正：一堆人在抢同一块蛋糕。逆：冲突被压下去了，但没解决。 |
| 权杖6 | 胜利、认可、凯旋 | 延迟成功、缺乏认可 | 正：你赢了，享受掌声。逆：赢了但没人知道，或还没赢。 |
| 权杖7 | 坚守立场、挑战、防御 | 放弃、不堪重负 | 正：你一个人对抗所有人，但你站得住。逆：撑不住了，考虑撤退。 |
| 权杖8 | 快速行动、消息、旅行 | 延迟、混乱、等待 | 正：事情快速推进，消息要来了。逆：一切都卡住了。 |
| 权杖9 | 坚持、疲惫但不放弃 | 固执、偏执、不知变通 | 正：你已经很累了，但再撑一下。逆：死撑着，但方向可能错了。 |
| 权杖10 | 负担过重、责任压身 | 放下负担、找人帮忙 | 正：你扛了太多，需要减负。逆：终于放下了，或找到了帮手。 |
| 权杖侍者 | 热情、好奇、新消息 | 冲动、缺乏经验 | 正：充满热情的新手。逆：热情有余，经验不足。 |
| 权杖骑士 | 冲劲、行动力、冒险 | 鲁莽、不计后果 | 正：全速前进，没有刹车。逆：冲过头了，需要踩刹车。 |
| 权杖王后 | 自信、热情、有魅力 | 嫉妒、控制欲 | 正：充满能量和魅力的女性。逆：用魅力操控他人。 |
| 权杖国王 | 领导力、远见、创业精神 | 独裁、冲动决策 | 正：有远见的领导者。逆：独断专行，不听建议。 |

## 圣杯（水象：情感/关系/直觉/内心）

| 牌名 | 正位含义 | 逆位含义 | 人话版 |
|-----|---------|---------|-------|
| 圣杯A | 新感情、情感开放、直觉 | 情感封闭、压抑感受 | 正：心门打开了，爱要来了。逆：你在关闭自己的心。 |
| 圣杯2 | 两情相悦、伙伴关系、承诺 | 关系不平衡、分离 | 正：双向奔赴，很好的开始。逆：一方付出更多，或关系出现裂缝。 |
| 圣杯3 | 庆祝、友情、社交、喜事 | 过度放纵、三角关系 | 正：好事值得庆祝，朋友在身边。逆：聚会变成问题，或有第三者。 |
| 圣杯4 | 冷漠、无聊、错失机会 | 重新投入、接受新事物 | 正：有机会在眼前，但你没注意到。逆：终于从冷漠中走出来了。 |
| 圣杯5 | 失落、悲伤、只看到损失 | 从悲伤中恢复、接受 | 正：你只盯着打翻的杯子，忘了还有两杯是满的。逆：开始走出来了。 |
| 圣杯6 | 怀旧、童年、纯真、过去 | 活在过去、无法前进 | 正：美好的回忆，或旧人重现。逆：你被过去困住了。 |
| 圣杯7 | 幻想、选择太多、迷惑 | 清醒、做出选择 | 正：你面对太多选项，但都是幻觉。逆：终于看清楚了，做出了选择。 |
| 圣杯8 | 离开、放弃、寻找更深意义 | 停留、逃避、不肯放手 | 正：你知道这里已经没有你想要的了，该走了。逆：明知道该走，还是走不了。 |
| 圣杯9 | 愿望实现、满足、享乐 | 自满、表面满足 | 正：你许的愿望实现了。逆：得到了，但没有想象中那么满足。 |
| 圣杯10 | 家庭幸福、情感圆满、和谐 | 家庭不和、破裂 | 正：幸福的家庭，情感的终点站。逆：表面和谐，内部有问题。 |
| 圣杯侍者 | 直觉、情感信息、敏感 | 情绪化、不成熟 | 正：直觉很准，有情感方面的消息。逆：情绪化，容易被感情左右。 |
| 圣杯骑士 | 浪漫、追求、情感行动 | 情感不稳定、花心 | 正：浪漫的追求者，感情进展。逆：花言巧语，不可靠。 |
| 圣杯王后 | 情感智慧、同理心、直觉强 | 情绪化、依赖 | 正：情感成熟，能理解他人。逆：情绪化，用感情操控他人。 |
| 圣杯国王 | 情感成熟、平衡、智慧 | 情感压抑、操控 | 正：情感上成熟稳重的人。逆：表面平静，内心压抑，或情感操控。 |

## 宝剑（风象：思维/冲突/真相/决断）

| 牌名 | 正位含义 | 逆位含义 | 人话版 |
|-----|---------|---------|-------|
| 宝剑A | 真相、清晰、突破、决断 | 混乱、谎言、错误决定 | 正：真相大白，切断迷雾。逆：你在用谎言欺骗自己或他人。 |
| 宝剑2 | 僵局、回避、难以决定 | 做出决定、信息浮现 | 正：你蒙着眼睛，不想面对现实。逆：终于睁开眼，做出了选择。 |
| 宝剑3 | 心碎、背叛、痛苦 | 从痛苦中恢复 | 正：被刺穿的心，但这是真实的痛苦，需要面对。逆：开始愈合了。 |
| 宝剑4 | 休息、恢复、暂时退出 | 无法休息、强撑 | 正：你需要休息，暂时退出没关系。逆：你该休息了，但停不下来。 |
| 宝剑5 | 冲突、胜利但代价大、不光彩的赢 | 和解、放下争斗 | 正：你赢了，但赢得不光彩，代价很大。逆：放下争斗，寻求和解。 |
| 宝剑6 | 过渡、离开困境、前往更好的地方 | 无法离开、延迟 | 正：你正在离开困境，前方更好。逆：想走但走不了。 |
| 宝剑7 | 欺骗、偷偷行动、策略 | 被揭穿、良心发现 | 正：有人在背后搞小动作，可能是你，也可能针对你。逆：阴谋被揭穿。 |
| 宝剑8 | 被困、自我限制、受害者心态 | 解脱、重获自由 | 正：你被困住了，但其实是你自己绑住自己的。逆：终于意识到可以离开了。 |
| 宝剑9 | 焦虑、噩梦、过度担忧 | 从焦虑中解脱 | 正：你在深夜反复想最坏的情况。逆：焦虑减轻，开始看到出路。 |
| 宝剑10 | 最坏的结局、背叛、结束 | 从谷底反弹 | 正：这是最坏的情况，但结束了就是结束了，可以重新开始。逆：谷底反弹，往上走了。 |
| 宝剑侍者 | 好奇、监视、信息收集 | 八卦、散布谣言 | 正：有人在观察你，或你需要多收集信息。逆：有人在散布谣言。 |
| 宝剑骑士 | 快速行动、直接、冲动 | 鲁莽、攻击性 | 正：快刀斩乱麻，直接行动。逆：冲动行事，伤人伤己。 |
| 宝剑王后 | 独立、清醒、直接 | 刻薄、冷漠 | 正：看透一切的女性，说话直接不绕弯。逆：用言语伤人，或过于冷漠。 |
| 宝剑国王 | 权威、公正、分析力强 | 独裁、冷酷 | 正：公正的权威人物，逻辑清晰。逆：用权力压人，或过于冷酷。 |

## 星币（土象：物质/金钱/现实/身体）

| 牌名 | 正位含义 | 逆位含义 | 人话版 |
|-----|---------|---------|-------|
| 星币A | 物质机遇、新财务开始、踏实 | 错失机会、财务不稳 | 正：有个实实在在的机会，抓住它。逆：机会在眼前，但你没抓住。 |
| 星币2 | 平衡、适应、多线并行 | 失衡、财务压力 | 正：你在玩杂耍，暂时平衡着。逆：杂耍的球要掉了。 |
| 星币3 | 技能、合作、工作成果 | 缺乏合作、粗心 | 正：团队合作，技能得到认可。逆：各自为政，工作质量下降。 |
| 星币4 | 守财、稳定、控制 | 吝啬、物质执念 | 正：你在保护自己的财富，没问题。逆：太抠了，或对物质过度执着。 |
| 星币5 | 财务困难、贫困、被排除在外 | 财务恢复、接受帮助 | 正：经济困难，感觉被世界遗弃。逆：开始走出困境，有人伸出援手。 |
| 星币6 | 给予与接受、慷慨、平衡 | 不平等、附加条件的给予 | 正：有人在帮助你，或你在帮助他人。逆：帮助带有条件，或关系不平等。 |
| 星币7 | 等待收获、评估、耐心 | 缺乏耐心、投资无回报 | 正：你种下的东西快要收获了，再等等。逆：等了很久，但没有回报。 |
| 星币8 | 专注技能、努力工作、学徒 | 缺乏专注、追求捷径 | 正：专注磨练技能，这是正确的路。逆：三心二意，或想走捷径。 |
| 星币9 | 独立、物质成就、自给自足 | 过度物质化、孤独 | 正：你靠自己的努力达到了物质上的成功。逆：有钱但孤独，或物质填不满内心。 |
| 星币10 | 财富传承、家族稳定、长期成功 | 家族问题、财务纠纷 | 正：长期稳定的财富，家族的成就。逆：家族内部有财务纠纷，或遗产问题。 |
| 星币侍者 | 学习、新机会、踏实 | 缺乏专注、好高骛远 | 正：踏实学习，机会在积累中。逆：想得多，做得少。 |
| 星币骑士 | 稳定、勤奋、可靠 | 停滞、过于保守 | 正：慢但稳，一步一个脚印。逆：太慢了，错过了时机。 |
| 星币王后 | 务实、滋养、物质稳定 | 过度物质化、控制 | 正：务实能干，把家管得井井有条。逆：用物质控制他人，或过度物质化。 |
| 星币国王 | 财富、稳定、商业成功 | 贪婪、固执 | 正：成功的商人，物质上的成就者。逆：贪婪，或用财富控制他人。 |
