## 合盘分析（比较盘 Synastry）

### 最重要的跨盘相位

| 相位组合 | 含义 | 人话版 |
|---------|-----|-------|
| A太阳合B月亮 | 灵魂伴侣感，互相滋养 | 你们天生就是一对，彼此理解 |
| A太阳对分B太阳 | 相互吸引但有张力 | 你们很不同，但正因如此吸引 |
| A金星合B火星 | 强烈的吸引力 | 化学反应极强，身体吸引力大 |
| A月亮合B月亮 | 情感共鸣 | 情感节奏一致，很舒服 |
| A土星合B太阳 | 土星方有约束感 | 土星方像父母，太阳方感到被限制 |
| A冥王合B金星 | 强烈迷恋，可能有控制动态 | 这段关系会彻底改变金星方的爱情观 |
| A木星合B太阳 | 木星方带来好运 | 木星方让太阳方感到幸运和扩张 |
| A土星四分B月亮 | 情感受限，有压力 | 月亮方在这段关系中感到情感压抑 |
| A天王合B太阳 | 突变和解放 | 天王方让太阳方经历突变，可能是解放也可能是混乱 |

### 合盘结论框架

```
【吸引力来源】：为什么两人会在一起（金星/火星相位）
【情感连接】：情感是否匹配（月亮相位）
【长期稳定性】：能否走远（土星/木星相位）
【主要挑战】：最大的摩擦点在哪里
【关系定位】：这段关系更像什么（激情/稳定/成长/业力）
【建议】：如何最大化这段关系的潜力
```

|-----|-----|-----|

### 流年解读优先级
1. 土星行运（最重要，代表现实压力和考验）
2. 木星行运（机会和扩张）
3. 天王星行运（突变和解放）
4. 行运过境本命太阳/月亮/上升（个人层面的重大影响）
5. 行运过境本命金星/火星（感情和行动层面）
